# Khohang - Taste of Thakali Website Design Brainstorm

## Design Approach Selected: **Himalayan Heritage Warmth**

### Design Movement
**Warm Minimalism with Cultural Authenticity** — A contemporary approach that honors Thakali heritage through earthy tones, hand-crafted elements, and deliberate negative space. Inspired by traditional Mustang architecture and modern hospitality design.

### Core Principles
1. **Authenticity First**: Every design element reflects genuine Thakali/Nepali culture—no generic "ethnic" aesthetics
2. **Warmth Through Color**: Use the brand's earthy palette (cream, gold, rust, charcoal) to evoke hearth, hospitality, and mountain tradition
3. **Hierarchy Through Restraint**: Let content breathe with generous whitespace; typography and imagery do the heavy lifting
4. **Craft & Texture**: Subtle details (hand-drawn dividers, natural shadows, organic spacing) suggest handmade quality

### Color Philosophy
- **Cream (#FFF3E0)**: Primary background—soft, inviting, like natural linen or aged parchment
- **Gold (#CC910D)**: Accent for highlights, calls-to-action, and cultural richness (echoes of temple ornaments)
- **Rust/Terracotta (#AD481C)**: Secondary accent for warmth and grounding (traditional Thakali architecture)
- **Charcoal (#2D2D33)**: Text and structural elements for contrast and legibility
- **Subtle warm grays**: Supporting text and borders for depth without coldness

**Emotional Intent**: Visitors should feel welcomed into a home, not a corporate venue. The palette evokes mountain hospitality, evening gatherings, and the comfort of shared meals.

### Layout Paradigm
- **Asymmetric & Organic**: Avoid rigid grids; use diagonal cuts, staggered sections, and flowing whitespace
- **Hero Section**: Full-width image of the garden/courtyard with overlay text—establishes the "khohang" concept immediately
- **Alternating Content Blocks**: Text on left, image on right (then reverse)—creates visual rhythm without monotony
- **Floating Elements**: Subtle offset cards, images, and text boxes create depth and movement
- **Generous Margins**: 40-60px padding around sections; breathing room is a design feature

### Signature Elements
1. **Hand-Drawn Dividers**: Organic SVG dividers between sections (wavy, angular, or inspired by Thakali geometric patterns)
2. **Circular Image Frames**: Thakali circular motifs (like the logo's mandala) used for key images—breaks rectangular monotony
3. **Warm Drop Shadows**: Soft, warm-tinted shadows (not pure black) to maintain the cozy aesthetic

### Interaction Philosophy
- **Subtle Hover States**: Gentle color shifts (cream → light gold), slight scale increases, smooth transitions
- **Smooth Scrolling**: Parallax effects on hero; fade-in animations as sections enter viewport
- **CTA Buttons**: Warm gold background with charcoal text; hover state adds subtle shadow and slight lift
- **Gallery Interactions**: Lightbox overlays with warm backdrop; smooth transitions between images

### Animation Guidelines
- **Entrance Animations**: Fade-in + slight upward movement (200-300ms) for sections as user scrolls
- **Hover Transitions**: 200-300ms ease-in-out for all interactive elements
- **Scroll Parallax**: Subtle (20-30px offset) on hero background image for depth
- **Loading States**: Gentle pulse or shimmer using the gold accent color
- **No Jarring Motion**: Animations should feel natural and restful, never aggressive or distracting

### Typography System
- **Display Font**: "Playfair Display" or "Cormorant Garamond" (serif, elegant, cultural weight)
  - Used for: Page titles, section headings, restaurant name
  - Weight: 600-700 for impact
  - Size: 48px (mobile) → 64px (desktop) for hero; 32-40px for section headings
  
- **Body Font**: "Lato" or "Open Sans" (sans-serif, warm, readable)
  - Used for: Body text, descriptions, menu items
  - Weight: 400 (regular), 500 (emphasis)
  - Size: 16px (mobile) → 18px (desktop) for body; 14px for supporting text
  
- **Accent Font**: "Poppins" (modern sans-serif for CTAs and labels)
  - Used for: Buttons, navigation, time/date info
  - Weight: 600
  - Size: 14-16px

- **Hierarchy**: 
  - H1: Display font, 64px, charcoal, 1.2 line-height
  - H2: Display font, 40px, rust accent, 1.3 line-height
  - H3: Body font, 24px, charcoal, 500 weight
  - Body: Body font, 18px, charcoal, 1.6 line-height
  - Small: Body font, 14px, warm gray, 1.5 line-height

## Implementation Notes
- All images sourced from Khohang's official social media (Facebook, Instagram, TripAdvisor, Google)
- Logo placed prominently in header and footer

- Accessibility: WCAG AA compliance, high contrast ratios, semantic HTML
